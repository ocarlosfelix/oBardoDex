# oBardoDex
Essa aplicação é uma Calculadora de Cachê projetada para auxiliar músicos e artístas na elaboração de orçamentos para apresentações artísticas.

### Tecnologias
Essa aplicação foi desenvolvida em HTML5, CSS3 e Javascript.

### Rodando o Projeto
1 - Clone o repositório;
2 - Execute o index.html em um navegador.
